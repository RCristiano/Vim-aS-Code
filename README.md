## Requirements
- NeoVim
- vim-plug
- ctags


## Instalation

### NeoVim

Move init.vim file and extensions folder to ~/.config/nvim/

```sh
mv init.vim ~/.config/nvim/
mv -r extensions ~/.config/nvim/
```

Then run:

```
:PlugInstall
```

# Vim aS Code

Vim with VSCode inspirations

## Prints

![print](vimascode.png)
 
## Requirements
- [vim-plug](https://github.com/junegunn/vim-plug/)
- ctags [universal-ctags](https://github.com/universal-ctags/ctags)
- python3 (python 2 is optional)
- pynvim (for python)


## Instalation

### NeoVim
```sh
mv init.vim ~/.config/init.vim
```

### Vim
```sh
mv init.vim ~/.vimrc
```
